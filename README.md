Documentación del proyecto.

Respuestas Teóricas
¿Para qué se utilizan los operadores lógicos en programación?
Se utilizan para combinar varias condiciones y tomar decisiones complejas en el código. Nos ayuda a evaluar si varias reglas se cumplen al mismo tiempo o si basta con que se cumpla al menos una para invertir el valor de una condición.

¿Por qué es importante declarar correctamente el tipo de dato de una variable?
Es importante ya que hace más eficiente utilizando la memoria de forma correcta y no usar espacio que no es necesario usar, también nos protege de cometer errores lógicos y define como reaccionará la variable conforme la utilidad que le estemos dando.

Justificación Tecnica.
Justificación del programa "Clasificacion de Edad":
Utilice if-else, porque al usarlo el programa deja de buscar en cuanto encuentra la primera condición verdadera y buscamos entre rangos con operadores logicos, también es más comodo en corregir errores por si el usuario ingresa un dato invalido.

Justificación del programa "Día de la Semana":
Utilice switch, porque no buscamos un rango sino un número exacto, de esa manera utilizar switch fue la mejor decición.

Justificación del programa "Verificación de Acceso":
Utilice if-else, porque tenía que evaluar dos variables diferentes, un switch no podría hacerlo, ya que funciona para evaluar una sola variable a la vez, no sería eficiente utilizarlo.
